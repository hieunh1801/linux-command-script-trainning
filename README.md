## Basic Command 
| Comand      | Description                                  |
|:------------|:---------------------------------------------|
| __pwd__     | show current path of directory               |
| __cd /__    | change directory to the root directory __/__ |
| __cd root__ | go to directory __root__                     |
| __ls__      | show folder in the current directory         |
